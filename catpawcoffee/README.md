在某个街道，有一家「猫爪咖啡屋」。
在里面有许多年轻漂亮的女孩子。
只要坐到她的对面，为她点一杯咖啡，你们就可以开始聊天。
在她喝完咖啡之前，如果你能俘获她的芳心，那她今晚就会属于你。

调情可以同时增加好感度和S欲，但是增加的数值比较低；

合理的在每回合选择行动，将好感度提升到200，S欲提升到100，接下来她就属于你了~
